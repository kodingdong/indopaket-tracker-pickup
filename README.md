# IndoPaket Tracker

Aplikasi web sederhana untuk melacak paket Indomaret dan mengelola jadwal pickup untuk mencegah return otomatis karena melewati batas waktu pengambilan (biasanya 5 hari).

## Fitur Utama (MVP)
- Dashboard melihat ringkasan status paket
- Tambah paket dengan resi, tanggal tiba, dan lokasi toko
- Perhitungan sisa hari otomatis dan indikator status (Aman, Segera, Besok, Hari Ini, Terlambat)
- Filter dan sorting paket (Berdasarkan urgensi/toko)

## Tech Stack
- Vanilla HTML/CSS/JS
- LocalStorage
- GitHub Pages

## Cara Menjalankan
1. Clone repository ini
2. Buka `index.html` di browser
3. Aplikasi siap digunakan
